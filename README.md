# spacegeame

a game simple space game written in c++ using SDL2

![screenshot](https://github.com/Ztirom45/SpaceGame/blob/main/img/Screenshots.png)

##### Features:
- Menu
- Gamemodes
- 5 Level
- different enemy types

##### Building(unix-like):	
- compile: `$sh build/compile.sh`
- run: `$./bin/main`
